# OrangeHRM Playwright Test Project

This is a beginner automation testing project using Playwright.

The project tests the OrangeHRM demo website.

---

## 🔗 Website Tested
https://opensource-demo.orangehrmlive.com

---

## 🛠 Tools Used
- Playwright
- JavaScript
- Node.js
- VS Code
- GitHub

---

## 📁 Project Structure
tests/
orangehrm/
login/
dashboard/
admin/
profile/


---

## 🧪 Test Cases Covered

### Login
- Valid login
- Invalid login
- Empty fields

### Dashboard
- Page loads after login
- Menu is visible

### Admin
- Open Admin page
- Search user
- Reset search
- Add user
- Search with no result

### Profile
- Change password validation

---

## ▶️ How to Run the Tests

### Install dependencies
npm install

---

## 📊 Report

Playwright generates a report after running tests.

The report shows:
- Passed tests
- Failed tests
- Error details
